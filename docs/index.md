# Mosaic Architecture

This site is the documentation portal for Mosaic architecture, design language, design system, and engineering guidance.

## Documentation Areas

- [Engineering](engineering/index.md) contains implementation-facing architecture and engineering guidance.
- [Design](design/index.md) contains Mosaic Design Language and Mosaic Design System specifications.

Each specification is maintained as a separate book. Chapters remain split into focused Markdown files so the source stays reviewable and the generated site remains navigable.

## Current Documentation

The current documentation set is draft material. Draft does not mean disposable: changes should be intentional, reviewed, and traceable.
