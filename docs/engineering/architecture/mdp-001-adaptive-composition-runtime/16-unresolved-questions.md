# Unresolved Questions

> **Proposal status:** Deferred and non-authoritative. This chapter preserves post-v1 research; it is not a Mosaic v1 requirement.

---

# Purpose

Deferral preserves uncertainty rather than converting provisional assumptions into standards.

MDP-001 requires renewed evidence before any of these questions can become accepted architecture.

---

# Solver Calibration

- Which v1 reference compositions provide representative training and comparison cases?
- How should pairwise stakeholder ranking translate into initial private weights?
- What candidate count and beam width remain safe under measured Presentation budgets?
- Which constraints must remain hard across every accessibility configuration?

---

# Spatial Model

- How many governed logical depth planes are perceptually useful?
- Which semantic changes justify a settled depth change?
- How should projected text clearance scale with typography and viewing distance?
- Which saliency sources are reliable enough to establish hard Airspace Reserves?

---

# Motion Model

- Which critically damped frequencies preserve immediate response across input methods?
- How should interruption and retargeting preserve velocity without visible instability?
- Which transitions benefit from Behavioural Cost beyond ordinary v1 component Motion Tokens?

---

# Reactivation Evidence

The proposal should not return to Active disposition until Mosaic has:

- production-quality v1 component compositions
- cross-extent visual regression fixtures
- measured component and Refraction performance
- validated semantic SDUI structures
- stakeholder-ranked alternative compositions
- an explicit non-v1 Roadmap horizon
